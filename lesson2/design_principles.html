Contrast

Footprint

footprintus

Footprints main page is a good example of how to use colors to create contrast. Because Footprint is a company that seeks to reduce plastic waste globally, naturally their main color is the color green. Their page uses green to provide contrast to the otherwise white background. Also, when white is used as a background, they opt to use green instead of black to put extra emphasis on going green. This is a great demonstration on how using contrast can help keep with a companies theme.

Alignment

BYU-Idaho

byu.edu

Notice how this page aligns everything in an orderly fasion? The BYU-Idaho site pages need to be easy for students to navigate and also present a professional decoration. In this case, heavy emphasis on alignment helps to accomplish this task. Every part of page is aligned to another element. It enforces a well organized look that makes it easier to digest the information displayed for the web user.

White Space

Visual Studio

visual studio

The primary use of this page is for any programmer to download Microsoft's Visual Studio application. There isn't a whole lot else going on in this page so naturally there is a lot of empty space. The excellent use of imagery and also the positioning of content elements helps turn an otherwise empty page that serves only a single purpose into a very visually appealing work of art that sells the professional focus and style that is being presented. This is done with white space, or in this case, a black background gradient that supplements the images and sharply transitions to a white background, providing some very neat contrast.
